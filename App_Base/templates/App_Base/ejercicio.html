Ejercicio a Realizar.

Ejercicio Django

El ejercicio debe realizarse con python django en su última versión, se debe utilizar python 3 y se deberá entregar un link con acceso al repositorio para poder hacer un clon del proyecto, el cual deberá contar con los requerimientos a instalar en la virtual para testear.

Sistema Logística

Un cliente necesita poder sistematizar los paquetes que le llegan al depósito, existen 3 elementos bien identificados en esta parte: PAQUETE, PLANILLA, ITEM Un paquete tiene

  • tracking que es único
  • dirección de destinatario
  • telefono de destinatario
  • nombre de destinatario
  • peso
  • altura
  • estado (Los estados pueden ser “en deposito”, “en distribución”)
  • cliente (modelarlo en una clase)
  • tipo de paquete (no lo carga el usuario lo tiene que tomar el sistema según el peso que fue cargado)
    • P peso menor 1000
    • M peso menor 3000
    • G todo lo demas

Una planilla tiene

  • número de planilla
  • fecha
  • item de planilla (uno o muchos paquetes)

    • *Estos ítems se verán en la pantalla de la planilla y se podrán agregar todos los necesarios en esta pantalla.

  • crear una acción que permita elegir una planilla y pasar todos los paquetes “en depósito” a “en distribución”

Un ítem de planilla tiene:

  • Un paquete
  • Una plantilla
  • una posición
  • un motivo de fallo (a futuro se desea hacer estadística sobre Motivos de Fallo)

El sistema tiene que permitir crear o modificar paquetes. Al entrar a una planilla se debe poder agregar items de planilla.

Agregar filtros y buscadores en las distintas pantallas que crea necesario. EXTRA: Una funcionalidad especial del sistema es que se debe permitir la carga masiva de paquetes a un recorrido (planillas), a partir de un CSV. (pantalla especial fuera del admin)
